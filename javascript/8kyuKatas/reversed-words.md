## Multiply

The code does not execute properly. Try to figure out why.

---

### Solution

```
function solution(str){
    let splitStr = str.split('');
    let reverseStr = splitStr.reverse();
    let joinStr = reverseStr.join('');
    return joinStr;
}
```
